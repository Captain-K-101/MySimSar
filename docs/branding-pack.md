# Branding Pack (Luxury GCC)

## Logo
- Wordmark with geometric Arabic-inspired accent; badge variant combining location pin + shield/check.
- Deliverables: monochrome + full-color, SVG + PNG; works on light/dark backgrounds; favicon derived from badge.

## Color Palette
- Primary: Deep Navy (#0B2545) — trust/luxury.
- Secondary: Desert Gold (#C89B3C) — premium highlight.
- Accent: Emerald (#1FA37A) — growth/approval.
- Neutrals: Obsidian (#0F172A), Warm Gray (#334155), Mist (#E5E7EB), White (#FFFFFF).
- States: Success (#16A34A), Warning (#F59E0B), Danger (#DC2626), Info (#0284C7).
- Guidance: maintain WCAG AA contrast; adjust tints/shades for dark mode while preserving hierarchy.

## Typography
- Headings: \"Sora\" or \"Playfair Display\" for luxe hero moments.
- Body/UI: \"Inter\" for readability; fallback system stack.
- Define scales: e.g., display 48/56, h1 32/40, h2 24/32, h3 20/28, body 16/24, small 14/20.

## Components & Tokens
- Buttons: primary (navy), secondary (outline navy/gold), tertiary (ghost); hover/active states.
- Badges: Verified, tier chips (Bronze/Silver/Gold/Platinum), score pill.
- Cards: soft shadow, 8–12px radius; use gold accent for highlights.
- Inputs: subtle borders, focus ring in emerald; error in danger color.
- Navigation: top bar + sidebar for admin; pill tabs for user/broker dashboards.
- Tailwind tokenization: colors mapped to palette, spacing 4/8/12/16/24/32, radii 6/10/12, shadows sm/md/lg tuned for light/dark.

## Imagery & Iconography
- Minimal line icons; avoid heavy fills.
- Photography: premium UAE skylines, interiors; muted overlay for text legibility.
- Map pin styling aligned to badge; verified badge consistent across UI, emails.
